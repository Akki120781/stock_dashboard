# Stock Dashboard - Recruitment Project

Welcome to the Stock Dashboard! This is a full-stack MERN application designed to let users search for stocks in real-time and manage their personal watchlists.

### Live Demo
- **Frontend** (Vercel): [https://stock-dashboard-three-chi.vercel.app/](https://stock-dashboard-three-chi.vercel.app/)
- **Backend API** (Render): [https://stock-dashboard-509t.onrender.com/](https://stock-dashboard-509t.onrender.com/)

### Key Features
- **Real-time Stock Search**: Integrated with Alpha Vantage (or similar) APIs for live data.
- **Watchlist Management**: Add and monitor your favorite stocks.
- **Modern UI**: Dark/Light mode, animations, glassmorphism, fully responsive design using Tailwind CSS.
- **Secure Authentication**: JWT-based authentication system built from scratch.

### Tech Stack
- MongoDB, Express.js, React.js, Node.js (MERN)
- Tailwind CSS v4 for styling
- Axios for API requests
